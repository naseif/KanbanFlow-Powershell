# Get-KBFDayOverview

*Grabs all time entries, groups them as report*

Sometimes you have to report what your day was like.
No problem if you used the kanbanflow timer features.

## Example Nr. 1
Get-KBFDayOverview -ApiToken $board -Day ((Get-Date).Date)
